Keyboard navigation for search results Key
Jumps down one entry in the list (if tapped in search box, jumps to first result) Down Arrow
Jumps up one entry in the list (if tapped at first result of first category, jumps to search box) Up Arrow
Jumps down a maximum of 5 entries in the list PageDown
Jumps up a maximum 5 entries in the list PageUp
Jumps to first result of next category (if there are results in than 1 categories, otherwise same as Down Arrow Key) Tab
Jumps to last result of previous category (if there are results in more than 1 category, otherwise same sas Up Arrow Key) SHIFT-Tab
 
The Enter key will select the element (same as clicking on the element).
Down/Up/PageDown/PageUp also jump categories when at end/start of list.
